# FIFA World Cup 2026 Winner Prediction using Machine Learning

## Overview

This project predicts the FIFA World Cup 2026 champion using Machine Learning. A Random Forest Classifier is trained on historical international football match data to predict match outcomes. The trained model is then used to simulate the knockout stage and predict the tournament winner.

---

## Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Random Forest Classification
- Model Evaluation
- Knockout Stage Simulation
- FIFA World Cup 2026 Champion Prediction

---

## Dataset

Historical international football match results containing:

- Home Team
- Away Team
- Home Score
- Away Score
- Tournament
- Neutral Venue

## Machine Learning Workflow

1. Load historical football match dataset
2. Clean the dataset
3. Perform Exploratory Data Analysis
4. Engineer team performance statistics
5. Train Random Forest Classifier
6. Evaluate model performance
7. Simulate FIFA World Cup 2026 Knockout Stage
8. Predict FIFA World Cup 2026 Champion
